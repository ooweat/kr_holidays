# kr_holidays
대한민국 공휴일 확인 / 영업일 확인

# 개요 
> 실무에서는 영업일에만 동작하도록 하는 프로세스가 꽤나 빈번하게 있다.
> Database 의 Holiday 테이블을 생성하여 작업을 진행하고 있으나 조금 더 가볍게 사용하기 위해
> 라이브러리로 구현해보았다.
> 
# 목표
> 1. 공통 Method 구현
> 2. 인터넷 망을 사용중일 경우, 공공포털 API를 통해 대한민국 공휴일을 return 받는다.
> 3. 인터넷이 존재하지 않을 경우, 고정된 날짜를 토대로 계산한다.
